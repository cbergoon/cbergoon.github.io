Senior Design Capstone Project: Nano-Studio

Nano-studio is an immersive and innovative educational platform for sudents, focusing primarily on nanotechnologies. The Nano-studio application features a 3D environment with Leap Motion controller integration and several components ranging from chemical structure visualizations to atomic scale microscopy simulations. The desktop application is written in C# with Unity 3D; the web application in JavaScript with Node.js and WebGL.

Nano-studio was developed under the guidance of Dr. Savas Kaya in conjunction with the NanoStudio Lab at Ohio University, a NSF funded project.
